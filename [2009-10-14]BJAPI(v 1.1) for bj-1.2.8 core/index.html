说明:

这里API实现的功能都是基于BJ这个全局对象来运作的。

强烈推荐使用带有firebug的firefox来看这个API。
(当然IE6.0+也可以，IE8没有测试。

这样就可以少考虑一个浏览器了。)

//目前只是支持具备id名称的节点，并没有写XHTML内置标签筛选[比如：BJ("div");或者 BJ("div p");]。

//如果第传入的是#加上idName,那么返回的仅仅是一个DOM节点了。

//举个例子: BJ扩展的addClass函数
BJ.extend("addClass", function(c){
    BJ(this.elem).each(function(){
        var oldClass = this.className.toString().split(" ");
        for (var i = 0; i < oldClass.length; i++) {
            if (oldClass[i] == c) { //如果新加的class是以前有的，那么就不加上去了
                return;
            }
        }
        var newClass = oldClass == "" ? c : oldClass.join(" ") + " " + c;
        this.className = newClass;
    });
	
	return BJ(this.elem);
});

ajax

说明:

这个是ajax的封装，里面内置了常用的函数。使得ajax写起来更方便。

url (string 需要请求的地址)
before(function 还没有完成的状态所触发的函数)
success(function 完成后所触发的函数)
error (function 发生错误时触发的函数)
checkStatus (boolean 确认status是否为200)
stop (function 中断ajax请求的函数)
time (number 超时的时间)
xml (boolean 返回的是不是xml)
type (string 请求的类型)
asyn (boolean ajax是异步还是同步)

用法: 
BJ.ajax({
	//必须
	//请求的地址，我这里只是放了个例子地址。
	url:"example.asp?id=3",
	
	//非必须
	//一般用在laoding，自己可以发挥想象。
	before:function(){
	  BJ("msg").html("loading....");
	},
	
	//非必须
	//如果加上了这句，就会判断status是否等于200，
	//如果不等于，那么就是弹出error，不运行success，并且告诉你status code。
	//如果status是等于200，那么就运行success。
	checkStatus:true,
	
	//非必须
	//默认是false
	xml:true,
	
	//非必须
	//默认是get
	type:"post",
	
	//非必须
	//默认是true(异步)
	asyn:false,
	
	//必须
	//msg 是通过url请求后传过来的数据，如果没有前面的xml参数的话，就是返回的是string。
	//如果xml设置为true的话，那么返回的是xhr对象。
	success:function(msg){
	  alert(msg);
	},
	
	//非必须
	//修改stop调用所需要的时间，默认是10秒，这里把它修改为8秒。
	time:8000,
	
	//非必须
	//这个默认是10秒，也就是说10秒后，还没有完成(即readyState!=4)。
	stop:function(){
	  alert("服务器超时，请稍后再试。");
	},
	
	//非必须
	//如果出错了，success就不会运行。这个出错是没有xhr对象
	error:function(){
	  alert("出错了~~~");
	}
});

BJ(”idName”).hide("fast",function(self){alert("哈哈，被隐藏了.")});(有动画效果,并有回调函数,self表示是当前的节点)

扩展的动画函数
animate_start(); 是动画的初始函数
animate_stop(); 是动画的停止函数
animate_continue(); 是动画的继续函数

用法: 

BJ("idName").animate_start({
//默认是"position",position表示node是属于绝对定位。
  margin是依靠变化margin来移动	(不是必须)
  type:"position"或者"margin",
  
//从什么位置开始(必须)
  from:{left:100,top:100},
 
//到什么位置结束(必须)
  to:{left:700,top:100},

//默认是400，越小速度就越快(不是必须)
  speed:800,

//默认是"Cubic.easeOut",动画的类型，名字结合上面的Tween类(不是必须)
  animateType:"Cubic.easeOut",
  
//动画结束的时候，调用回调函数(不是必须)
  callback:function(){//do someting....}
});

//只有strat了，才能stop
BJ("idName").animate_stop(function(){//do someting....});

//只有stop了，才能continue
BJ("idName").animate_continue(function(){//do someting....});

drag

说明:

拖拽函数。

用法: 
BJ.drag({
  node:'idName' or node,     //(必须)	拖动的是那个元素
  
  hand:'idName' or node,     //(非必须)	鼠标放在哪里拖动
  
  onlyX,                     //(非必须)	为true则只能x方向
  
  onlyY,                     //(非必须)	为true则只能y方向
  
  opacity,                   //(非必须)	取100内的值，这样移动的时候会变为透明
  
  mouseup_callback(node),    //(非必须)	鼠标释放后的回调函数
  
  mousemove_callback(node),  //(非必须)	鼠标移动时回调函数
  
  mousedown_callback(node)   //(非必须)	鼠标按下去时回调函数
});

说明:

节点属性值的有关操作。
BJ('idName').attr(name, value);
@name 为属性的名字
@value 如果没有定义，则表示取name属性的值；如果定义了，则是为name属性只赋于value，并返回BJ对象

如果错误则返回空字符串。

用法: 
BJ('idName').attr("alt");
BJ('idName').attr("alt","logo");

var ps=BJ("div").children();
//把div下面的p的class全部设置为love
BJ(ps).attr("class","love");
//把div下面的p的class相应的设置为"love","hate","like",一一对应。数量一定要满足数组的长度
