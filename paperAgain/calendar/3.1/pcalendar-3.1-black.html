PCalendar Examples

默认情况（日期可以全部选择，中文界面，格式是：yyyy-mm-dd）

PCalendar({
  'node':'date1'
});

PCalendar({
  'node':'date2',
  'callback':function(s){
    document.getElementById('pcalendar-w').innerHTML=s;
  },
  'clearCallback':function(){
    document.getElementById('pcalendar-w').innerHTML='';
  }
});
不是输入框


设置了（日期以今天为分界线，英文界面，日历位置出现在右边，日期格式）

PCalendar({
  'node':'date3',
  'allValide':false,
  'lang':'en',
  'position':'right',
  'format':'yyyy/m/d'
});

设置了（关联）

PCalendar({
  'node':'date4',
  'endDateNode':'date5'
});
PCalendar({
  'node':'date5',
  'beginDateNode':'date4'
});

设置了（自定义的日期范围）

PCalendar({
  'node':'date6',
   //这里设置：[2011,6,4]就是指6月4号，而不是指7月4号（不是从0开始）
  'beginDate':[2011,6,4],
  'endDate':[2012,3,2]
});

PCalendar({
  'node':'date7',
   //只是设置了：beginDate。当然你也可以单独设置endDate
   //设置的内容不能为空
  'beginDate':[2011,6,4]
});
